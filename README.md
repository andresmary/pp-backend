## Descripción
Api para gestionar un servicio de estimación de tareas (Planning Poker).

## Instalación
```bash
$  yarn  install
```

## Ejecutar la aplicación
```bash
# development
$  yarn  run  start
# watch mode
$  yarn  run  start:dev
# production mode
$  yarn  run  start:prod
```

## Testeo
```bash
# unit tests
$  yarn  run  test
# e2e tests
$  yarn  run  test:e2e
# test coverage
$  yarn  run  test:cov
```
